ğŸ“Š Sorting Algorithms Implemented

ğŸ”¹ Bubble Sort

Bubble Sort works by repeatedly comparing adjacent elements and swapping them if they are in the wrong order. This process continues until the entire list is sorted.

ğŸ”¹ Insertion Sort

Insertion Sort builds a sorted array one element at a time by placing each element in its correct position relative to the already sorted portion of the list.

ğŸ”¹ Counting Sort

Counting Sort is a non-comparative algorithm that determines the position of each element by counting occurrences and placing elements accordingly. It is most effective for a limited range of integers.

ğŸ”¹ Radix Sort

Radix Sort sorts numbers by processing individual digits from the least significant to the most significant. It relies on Counting Sort as a subroutine to handle digit placement.

ğŸ”¹ Bucket Sort

Bucket Sort distributes elements into different buckets based on their values. Each bucket is then sorted individually, and the sorted elements are merged back together.
